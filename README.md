# 🌐 André's Portfolio

A personal portfolio website built for André, a friend and one of the best full stack developers I know. He gave me the opportunity to build his portfolio so I could improve my design and frontend skills.

🔗 **Live site:** https://arrz.tudu.dev/

---

## 💡 About

This portfolio was built to help André present himself and his work to both recruiters and fellow developers. The goal was to create something that felt personal and professional at the same time.

---

## 🧠 Thought Process & Decisions

This was a collaborative project in spirit. I built it, but André made most of the design choices.

A few things I kept in mind throughout:

- **Designed around the person, not the template.** Rather than dropping André's content into a generic layout, I build something more "arcade" vibes, with a tone of orange because that's his favorite color.
- **Scroll-controlled project stacking.** The most challenging part of the project was the highlighted projects section. It uses scroll-driven animations to simulate computer windows stacking on top of each other. Each project card layers in as the user scrolls, giving it a tactile, almost OS-like feel that matches the arcade aesthetic.
